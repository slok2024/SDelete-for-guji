SDelete数据粉碎终端 是一款基于微软 Sysinternals 核心工具 SDelete 开发的高级数据安全粉碎终端。本项目使用python专为 Windows 7 (32位/64位) 系统进行了底层架构优化，彻底摆脱了对系统残缺 COM 组件和外壳（Shell）的依赖，实现真正绿色、无依赖的数据彻底销毁。

使用- 算法：DoD 5220.22-M（3遍覆写）
